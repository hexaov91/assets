隱私權政策

資料收集
MRVN_Bot 收集並處理其功能所需的最少數據，例如使用者名稱、使用者 ID 和使用者活動（例如，加入或離開頻道）。該資料僅用於執行機器人功能，除了記錄使用者ID對應的偏好設置外，馬文不會儲存這些資訊。

使用第三方 API
MRVN_Bot 與遊戲 API 互動以檢索玩家資訊。當您使用涉及這些API的命令時，遊戲帳戶名稱可能會被暫時處理以提供相關資料。在查詢和回應所需的時間之外，不會共用或儲存此資訊。

資料共享
MRVN_Bot 不與第三方共享任何個人資料。機器人在 Discord 的基礎設施內運行，資料根據其隱私權政策保留在 Discord 伺服器上。

使用者控制
您可以隨時從伺服器移除 MRVN，這將立即終止與您的伺服器關聯的所有資料處理和權限。

安全
MRVN_Bot 採取合理措施保護您的數據。若發現任何安全漏洞，請向機器人管理員報告任何安全問題或未經授權的資料使用情況。
